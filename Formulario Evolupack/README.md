# Formulario web · Evolupack Discovery Express

Formulario listo para compartir, basado en el Discovery Express de Evolupack. Ábrelo localmente con `index.html` para revisarlo.

## Para recibir las respuestas

Una página estática no puede guardar respuestas por sí sola. Para activarlo, elige un servicio de recepción (por ejemplo Formspree, Netlify Forms o un formulario de Google) y coloca su URL de envío en el atributo `action` de la etiqueta `<form>` en `index.html`.

Ejemplo con Formspree: `action="https://formspree.io/f/TU_IDENTIFICADOR"`.

Después publícalo en el alojamiento que prefieras y comparte la dirección web con tus clientes.
